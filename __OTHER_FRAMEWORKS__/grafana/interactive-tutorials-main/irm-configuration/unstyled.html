Grafana IRM Setup and Configuration

This tutorial guides you through the process of setting up Grafana IRM for on-call notifications, including on-call schedules, escalation chains, and integrations.

For a more detailed walkthrough, including a summary of the value and components of IRM, please see the Configure Grafana IRM Learning Journey.

Section 1: Create an On-call Schedule

In this milestone, you create a basic schedule that uses a single rotation with team members taking turns to be on-call. This schedule lets you route alerts sent to your escalation chain based on the current on-call assignment.

Schedules provide a structured way to manage team on-call coverage and automate shift handoffs. When used in escalation chains IRM resolves who is on-call for a schedule and sends alert notifications to the appropriate users.

  • Navigate to Schedules in the IRM navigation menu. Click to create a new schedule. Select Set up on-call rotation schedule. Enter a name for the schedule (for example, Production SRE). Click Create Schedule. Create a new, empty on-call schedule.
  • Click to create a new rotation. If this button is disabled, you can skip and proceed to the next step. Click to add users to this rotation. Select at least one user. Create the rotation. Create an on-call rotation for the schedule.

    • Section 2: Create an Escalation Chain

    In this milestone, you create a basic escalation chain as the next step of your alert flow.

    Escalation chains define a sequence of notification steps for Alert groups: who gets notified, when they get notified, and what to do if an Alert group remains unacknowledged.
    A well-designed escalation chain ensures timely attention from the right people while avoiding unnecessary noise.

  • Navigate to Escalation chains in the IRM navigation menu. Click to create a new escalation chain. Enter a descriptive name for the escalation chain (for example, Production - Default). Click to create the escalation chain. Create a new escalation chain.
  • Click to add your first escalation step. Add a notification step to notify users from the on-call schedule. Select the on-call schedule you previously created. Add a second step to the escalation chain. Add a wait step, and configure a wait time of 5 minutes. Finally, add a step to notify users in case of escalation. Select the user(s) you want to escalate to. Configure steps for your escalation chain.

    • Section 3: Connect Grafana Alerting as an Integration

    In this milestone, you connect the Grafana Alerting integration for IRM. This integration is the foundation of your IRM setup. It allows you to send Grafana Cloud alert rules to IRM for automatic grouping, routing, escalation, and notification.

  • Navigate to Integrations in the IRM navigation menu. Click to create a new integration. Select Grafana Alerting from the list. Enter a descriptive name (for example, Production SRE alerts). Select to create a new contact point for this integration. Enter a descriptive contact point name (for example, Production IRM). Click Create Integration. Create a new Grafana Alerting integration.
  • Select the first route. Select the dropdown to choose an escalation chain. Select your escalation chain. Associate the integration with your escalation chain.
  • Click to send a demo alert. Confirm with Send alert. View your demo alert from the Alert groups page. Test your integration by sending a demo alert.
